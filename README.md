# 🛒 Daraz E-Commerce Sales Dashboard

## 📌 Project Overview
A complete Power BI dashboard built on 
simulated Daraz e-commerce data.

## 🛠️ Tools Used
- Power BI Desktop
- Power Query
- DAX

## 📊 Dashboard Pages
1. Executive Overview
2. Sales Analysis
3. Order Analysis
4. Seller Performance

## 🔍 Data Cleaning Steps
- Removed duplicates
- Replaced missing values 
- Fixed inconsistent city/category names
- Removed impossible dates
- Fixed negative prices

## 📈 Key Insights
- Electronics = 26% of total revenue
- 60% delivery success rate
- 21% customers use Cash on Delivery
- Islamabad is top revenue city

## ⚠️ Note
Dataset is synthetically generated 
for learning purposes only.
